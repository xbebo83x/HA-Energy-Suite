\# Logo



This folder contains the official visual identity of \*\*HA Energy Suite\*\*.



\## Files



\- `logo.png` → Official project logo

\- `logo-dark.png` → Logo optimized for dark backgrounds \*(future)\*

\- `logo-light.png` → Logo optimized for light backgrounds \*(future)\*

\- `logo-icon.png` → Icon-only version \*(future)\*

\- `logo.svg` → Vector version \*(future)\*



The logo is used across the GitHub repository, documentation, releases and future HACS integration.
